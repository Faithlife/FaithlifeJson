# JsonPatch.Apply method

Applies the patch to the specified JSON token.

```csharp
public JToken Apply(JToken before)
```

| parameter | description |
| --- | --- |
| before | The JSON token. |

## Return Value

The result of applying the patch to the JSON token.

## Remarks

To save time and memory, the JSON tokens in the patch are NOT cloned when applied to the result, and so should not be modified after the patch is applied. The 'before' token is not modified when applying the patch.
